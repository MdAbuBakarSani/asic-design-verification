# Problem 08: Write a Simple Bus Protocol Using `uvm_put_port`, `uvm_get_port`, and Two TLM FIFOs

## Problem Statement:

Write a simple **bus protocol** where:

- The **driver** sends a **request** using a **`uvm_put_port`**
- The **DUT** replies using a **`uvm_get_port`**
- Use **two TLM FIFOs** between the **driver** and the **DUT**:
  - one FIFO for **request**
  - one FIFO for **response**
