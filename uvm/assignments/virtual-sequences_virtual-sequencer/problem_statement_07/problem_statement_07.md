# Problem 07: Extend the Base Virtual Sequence to Create a `priority_virtual_sequence`

## Problem Statement:

Extend the **base virtual sequence** to create a **`priority_virtual_sequence`** that schedules child sequences based on an assigned **priority field**.

- Create a new virtual sequence named **`priority_virtual_sequence`**
- Extend it from the **base virtual sequence**
- Assign a **priority field** to the child sequences
- Sort the sequence handles based on **priority**
- Start the sequences only after sorting
- Higher-priority or lower-priority execution order should be controlled through the sorting logic
