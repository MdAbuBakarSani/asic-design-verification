## Problem Statement:

- Parse this argument in the **test** using **`uvm_cmdline_proc`**
- Set the parsed mode in **`uvm_config_db`**
- Make the **agent** adjust its mode accordingly
- Log whether the agent is operating in **active** mode or **passive** mode
