# Problem 11: Create a Single Agent That Switches from Active to Passive Based on Simulation Time

## Problem Statement:

Create a **single agent** that can act as either **active** or **passive** based on **simulation time**.

- For the first **200 ns**, the agent should act as **active**
- During this active period, it should **generate traffic**
- After **200 ns**, the agent should switch to **passive mode**
- In passive mode, it should **disable the driver** and **only monitor**
- Use **UVM phase objection management** and **time checks** to implement this behavior
