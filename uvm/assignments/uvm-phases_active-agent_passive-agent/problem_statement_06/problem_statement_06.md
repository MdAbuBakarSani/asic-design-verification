# Problem 06: Design an Agent That Switches from Active to Passive Dynamically

## Problem Statement:

Design an **agent** that switches from **active** to **passive** dynamically using a control flag.

- After a specific number of transactions, disable the **driver**
- Switch the agent to **passive mode**
- Implement this behavior inside the **driver**
- Use **`uvm_phase::raise_objection()`** and **`drop_objection()`** for clean transition handling
