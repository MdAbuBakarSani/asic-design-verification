# SystemVerilog Assignments - Showcase of Core Language Concepts 🎯

This folder contains assignments and problems related to **SystemVerilog**. The solutions here serve as a **demonstration of my understanding** of fundamental concepts in **SystemVerilog**.

--------

# 🔍 Purpose
The primary goal of this repository is to **showcase** the flow and structure of solving problems using **SystemVerilog** constructs and features. While the solutions may not represent **real-world, optimized designs**, they are intended to illustrate the **basic understanding** of how to approach and implement different concepts in **SystemVerilog**.

--------

# 🛠️ Approach
Each folder corresponds to a **different SystemVerilog topic**, and within each topic, the problems are presented along with the **simplified solutions**. The following topics are covered:

- **Verilog Operators**
- **Data Types**
- **Arrays**
- **Queues**
- **Mailbox**
- **Semaphores**
- **OOP Concepts**
- **Randomization & Constraints**
- **Interface**
- **Virtual Interface**
- **Fork Join**

--------

# 📂 How the Problems are Organized
- **Problem Statements**: Each problem is followed by a clear statement explaining the task.
- **Solutions**: The solutions demonstrate how to approach each problem using **SystemVerilog** features, with emphasis on **logic**, **syntax**, and **behavior**.
- **Test Cases**: When applicable, simple test cases are included to validate the functionality.

**Note**: These solutions are **educational demonstrations** and are intended to help me showcase the flow of solving SystemVerilog problems, rather than real-world, production-level designs. ⚙️

--------

# 📌 How to Use
1. Navigate to the respective topic folder (e.g., `Randomization_Constraints/`).
2. Inside each folder, you'll find:
   - **Problem statement files** (e.g., `problem_01.md`)
   - **Solution files** (e.g., `solution.sv`)
