# Problem 04: Implement a Priority-Based Message Passing System Using a Mailbox

## Problem Statement:

Implement a **priority-based message passing system** using a **SystemVerilog mailbox**.

- Messages should carry both **data** and **priority**
- Higher-priority messages should be handled before lower-priority ones
- Use a mailbox as the communication mechanism
- Display the order in which messages are processed
